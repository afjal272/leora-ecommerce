# 🛒 Leora — E-Commerce Platform

🔗 Live Demo: https://leora-ecommerce.vercel.app/

A full-stack eCommerce application built to simulate real-world business flow — from authentication to order management.

---

## 🚀 Core Features

### 🔐 Authentication
- OTP-based login  
- JWT authentication  
- Role-based access (User / Admin)

### 📦 Product System
- Admin-controlled product management  
- Dynamic product rendering from backend  

### 🛒 Cart & Checkout
- Add to cart with quantity control  
- Address selection  
- Seamless checkout flow  

### 📊 Orders
- Order creation & persistence  
- Order history tracking  
- Status-based lifecycle  

---

## ⚙️ Tech Stack

**Frontend**
- Next.js (App Router)  
- TypeScript  
- Tailwind CSS  
- Zustand  

**Backend**
- Node.js  
- Express.js  
- Prisma ORM  
- PostgreSQL
